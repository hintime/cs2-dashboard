# 任务总结：供需异动显示变化量

**时间**: 2026-04-19 17:39
**状态**: ✅ 已完成

---

## 目标
在「供需异动」中显示在售/求购的变化量（增加或减少了多少）。

---

## 实现方案

由于 CSQAQ API 只返回当前值，没有变化量字段，需要本地计算：

1. **历史数据存储**: 创建 `history.json` 保存上一周期的数据
2. **变化量计算**: `变化量 = 当前值 - 上周期值`
3. **显示格式**: `求购 671 (+10) / 在售 776 (-5)`

---

## 修改文件

| 文件 | 修改内容 |
|------|----------|
| `update.py` | 添加历史数据加载、变化量计算、历史保存逻辑 |
| `index.html` | 显示变化量（sell_chg, buy_chg）|
| `history.json` | 新增，保存历史数据 |

---

## 新增字段

```json
{
  "id": 34,
  "name": "双持贝瑞塔 | 雇佣兵 (久经沙场)",
  "buff_sell": 776,
  "buff_buy": 671,
  "sell_chg": -5,   // 在售变化量
  "buy_chg": 10     // 求购变化量
}
```

---

## 效果

- **首次运行**: 变化量为 0（无历史数据）
- **后续运行**: 显示实际变化量
- **显示格式**: `求购 671 (+10) / 在售 776 (-5)`
